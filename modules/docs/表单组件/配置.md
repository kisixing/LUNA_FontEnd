# 配置

> 表单组件分默认组件、业务组件，分别以`type`和`x-components`区分
>
> 容器组件[inspection_card](#/A8IbIE/6OSGSLI6U2)
>
> 减少信息噪音，降低误操作概率，最大化提升表单输入效率，所以，表单布局是表单解决
> 方案中很重要的一部分
